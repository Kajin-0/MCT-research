# Phase 1C issue scope

This file mirrors the authorized Phase 1C issue scope so the research gate remains version controlled.

## Objective

Resolve HgCdTe carrier statistics, source-controlled material parameters, deterministic boundary conditions, numerical architecture, and the final Phase 2 authorization decision.

## Acceptance criteria

1. Populate a source-controlled parameter table for band structure, statistics, permittivity, mobility, traps, optics, and contacts.
2. Quantify the error of the nondegenerate carrier and Einstein approximations over the intended domain.
3. Define ideal, finite-exchange, interface-state, blocking, and effective-`S` deterministic contact models.
4. Define fixed-voltage, fixed-current, and finite-load terminal ensembles.
5. Specify a conservative one-dimensional finite-volume formulation and independent verification route.
6. Specify executable Phase 2 analytical-limit, conservation, and convergence tests.
7. Record a final Phase 1 proceed, reframe, or terminate decision.

## Exclusions

- production simulations;
- broad parameter sweeps;
- stochastic solver implementation;
- manuscript drafting;
- novelty or predictive HgCdTe claims.
