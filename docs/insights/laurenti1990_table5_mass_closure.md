# Laurenti 1990 Table V effective-mass closure

## Question

Laurenti Table V lists conduction-band effective mass as a function of composition and temperature. Does it provide independent evidence for a temperature-dependent Kane matrix element, or is the mass variation generated by the fitted gap inside the paper's three-band model?

## Source status

Tables IV and V are not experimental datasets.

- Table IV contains gap values calculated from Laurenti equation 7.
- Table V contains masses calculated from a simplified three-band k.p model and virtual-crystal interpolation.
- The paper states that alloy lattice constant and polarity were linearly extrapolated.

This is therefore an internal closure test of historical model outputs.

## Diagnostic inversion

For the simplified three-band relation,

```text
m0/m* = 1 + P2/3 [2/Eg + 1/(Eg + Delta)].
```

Each printed Table IV/V pair can be inverted:

```text
P2 = 3(m0/m* - 1) / [2/Eg + 1/(Eg + Delta)].
```

The spin-orbit splitting is linearly interpolated between the Table VII endpoints:

```text
Delta_HgTe = 1.08 eV
Delta_CdTe = 0.927 eV.
```

The test is restricted to `0.50 <= x <= 1.00`, where rounding is less singular than near the zero-gap regime.

## Result

At each composition, infer `P2` independently from all six temperature rows, take the composition mean, and reconstruct every printed mass.

| Metric across 66 rows | Result |
|---|---:|
| maximum within-composition `P2` coefficient of variation | 0.783% |
| mass RMSE | `2.730e-4 m0` |
| mass mean absolute error | `2.274e-4 m0` |
| maximum mass error | `4.930e-4 m0` |

Table V is printed to `0.001 m0`. The maximum closure error is below half of the final printed increment.

Representative inferred composition means are:

| x | mean `P2` (eV) | within-x CV |
|---:|---:|---:|
| 0.50 | 19.7593 | 0.119% |
| 0.60 | 19.3559 | 0.700% |
| 0.70 | 19.1687 | 0.588% |
| 0.80 | 18.9566 | 0.671% |
| 0.90 | 18.6963 | 0.402% |
| 1.00 | 18.5221 | 0.333% |

The apparent scatter is compatible with the masses being rounded to `0.001 m0`.

## Endpoint cross-check

A linear diagnostic through the composition-mean inferred values gives

```text
P2(x) = 20.8482 - 2.36495 x  eV.
```

At CdTe this predicts `18.4832 eV`, agreeing with the Table VII three-band value `18.5 eV` to about `0.017 eV`.

This linear relation is only a diagnostic over `0.50 <= x <= 1.00`, not a proposed alloy law.

## Decision

```text
To Table V's printed precision, its temperature-dependent mass is generated
by Eg(x,T) with composition-fixed three-band coupling.
```

Consequently:

1. Table V is not independent experimental evidence for `P(T)`.
2. Its mass-temperature trend should not be counted separately from Laurenti equation 7 when assessing prior art.
3. Reproducing `m*(T)` through a changing gap and fixed coupling does not exceed this historical model.
4. A differentiating result requires independently resolved non-gap Hamiltonian motion or a held-out observable that cannot be reproduced by gap-only mass scaling.

## Limitations

- Both input tables were generated by the same paper's model.
- Alloy spin-orbit values are not printed and are represented by linear VCA interpolation.
- The mass rounding controls the inferred `P2` scatter.
- The zero-gap and inverted regimes are intentionally excluded.
- No claim is made about complete eight-band parameters `P8`, `P7`, `F`, or the Luttinger invariants.
