# CdTe quadratic Kane residual localization

## Status

The static CdTe finite-k smoke supports the linear two-`P` sector but rejects the conventional quadratic span generated by `F`, `gamma1`, `gamma2`, and `gamma3`. This note localizes that failure using the immutable PR #82 evidence. No new electronic-structure calculation is used or authorized.

## Question

The failed quadratic closure could have resulted from:

1. finite-radius contamination at the chosen `h = 0.01 A^-1`;
2. a simple low-rank defect that could be repaired by one additional matrix term;
3. broader missing matrix structure in the energy-independent eight-state Hamiltonian.

For each direction, define the Richardson-extrapolated quadratic coefficient

```text
C2(n) = lim_{h -> 0} [H(+h n) + H(-h n) - 2 H(0)] / (2 h^2)
```

and the conventional Kane residual

```text
R2(n) = C2_DFT(n) - C2_Kane(n).
```

The conventional parameters are fit only on `[001]` and `[111]`. `[110]` remains unused during fitting.

## Radius diagnosis

| Direction | Quadratic residual | Richardson correction | Correction / residual |
|---|---:|---:|---:|
| `[001]` | 7.3048 eV A^2 | 0.01357 eV A^2 | 0.1858% |
| `[111]` | 8.4578 eV A^2 | 0.00520 eV A^2 | 0.0615% |
| `[110]` | 8.1847 eV A^2 | 0.00930 eV A^2 | 0.1137% |
| Aggregate | 13.8523 eV A^2 | 0.01725 eV A^2 | 0.1246% |

The correction needed to remove the leading finite-radius error is approximately three orders of magnitude smaller than the residual. The approximately 49% quadratic failure is therefore not plausibly explained by the present `h` versus `h/2` difference.

## State-space rank

The aggregate residual singular values are

```text
6.6735, 6.6735, 5.2204, 5.2204,
4.1990, 4.1989, 2.5540, 2.5539 eV A^2.
```

All eight exceed 10% of the leading singular value. Direction-specific significant ranks at the same cutoff are:

| Direction | Significant rank |
|---|---:|
| `[001]` | 6 |
| `[111]` | 8 |
| `[110]` | 6 |

The near-paired singular values are consistent with the preserved time-reversal/Kramers structure. The defect is not a rank-one or rank-two state-space correction.

This does **not** prove that many independent symmetry invariants are required: one invariant can generate a full-rank matrix. It does rule out treating the failure as one isolated eigenstate or one small matrix element.

## Block localization

Fraction of total residual Frobenius energy:

| Block | `[001]` | `[111]` | `[110]` |
|---|---:|---:|---:|
| `Gamma6-Gamma6` | <1e-9% | <1e-9% | <1e-9% |
| `Gamma6-Gamma8` | 23.36% | 15.31% | 16.91% |
| `Gamma6-Gamma7` | <1e-7% | 27.59% | 22.10% |
| `Gamma8-Gamma8` | 45.54% | 33.92% | 36.23% |
| `Gamma8-Gamma7` | 19.99% | 14.89% | 15.90% |
| `Gamma7-Gamma7` | 11.12% | 8.29% | 8.86% |

The conventional conduction diagonal curvature is already closed: the `Gamma6-Gamma6` residual is numerically negligible. The failure is concentrated in the valence/split-off sector and in even-`k` inter-irrep blocks.

At a 5% energy-fraction cutoff, four blocks contribute for `[001]` and five blocks contribute for `[111]` and `[110]`. The residual therefore has broad block support rather than one isolated omitted coupling.

## Decision

The evidence supports the following conclusions:

1. **Finite-radius contamination is strongly disfavored.**
2. **The residual is not low-rank in state space.**
3. **The `Gamma6` diagonal quadratic term is already adequately represented.**
4. **The missing structure lies primarily in the valence and inter-irrep quadratic sectors.**
5. **The conventional `F`, `gamma1`, `gamma2`, and `gamma3` values remain rejected.**

## Next authorized gate

Generate the complete `Td`-covariant, time-reversal-even matrix basis that is quadratic in `k` for the fixed Novik

```text
Gamma6 + Gamma8 + Gamma7
```

representation. Then:

1. separate the conventional Kane span from the additional allowed span;
2. fit the additional span to `[001]` and `[111]` only;
3. retain `[110]` as a true holdout;
4. report design rank, conditioning, and the minimum number of additional identifiable combinations;
5. accept an extension only if both training and holdout residuals fall below the declared gate without severe conditioning.

No denser k mesh, additional bands, new functional, phonon, AHC, HgTe, or alloy calculation is authorized by this result.
