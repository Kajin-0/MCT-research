# Derivation 020: cross-scale covariance of variance estimates from one raster

## 1. Multiresolution maps share one material realization

Suppose the same spatial field is measured at probe scales

\[
s_1,\ldots,s_m.
\]

At scale \(i\), collect the values at \(n\) common sample centers into

\[
\mathbf y_i\in\mathbb R^n.
\]

The maps are jointly Gaussian under the declared Gaussian random-field model. Define the block covariances

\[
C_{ij}
=
\operatorname{Cov}(\mathbf y_i,\mathbf y_j).
\]

For Gaussian material covariance matrix \(\Lambda\), isotropic Gaussian probe scales \(s_i,s_j\), and sample-center displacement \(\Delta\mathbf r\), each block element follows the merged filtered-map theorem:

\[
[C_{ij}]_{ab}
=
A\sqrt{
\frac{\det\Lambda}
{\det[\Lambda+(s_i^2+s_j^2)I]}
}
\exp\left[
-\frac12\Delta\mathbf r_{ab}^{T}
[\Lambda+(s_i^2+s_j^2)I]^{-1}
\Delta\mathbf r_{ab}
\right].
\]

The off-diagonal blocks are not zero. Measuring several resolutions on one specimen region does not produce independent scale observations.

## 2. Naive variance estimators

Let

\[
P=I-\frac1n\mathbf1\mathbf1^T
\]

be the centering projector. The ordinary map variance at scale \(i\) is

\[
q_i
=
\frac{\mathbf y_i^TP\mathbf y_i}{n-1}.
\]

This estimator is unbiased for independent, equal-variance samples. It is generally biased for a finite correlated raster.

The expectation is

\[
\boxed{
\mu_i
\equiv
\mathbb E[q_i]
=
\frac{\operatorname{tr}(PC_{ii})}{n-1}.
}
\]

Let the marginal filtered variance be

\[
V_i=\frac1n\operatorname{tr}(C_{ii}).
\]

The deterministic bias factor is

\[
\boxed{
b_i=\frac{\mu_i}{V_i}.
}
\]

A known model-conditioned correction would use \(q_i/b_i\), but this correction does not create independent information.

## 3. Exact cross-scale quadratic-form covariance

For jointly zero-mean Gaussian vectors \(\mathbf x,\mathbf z\), symmetric matrices \(A,B\), and cross covariance \(C_{xz}\), Isserlis' theorem gives

\[
\operatorname{Cov}
(\mathbf x^TA\mathbf x,\mathbf z^TB\mathbf z)
=
2\operatorname{tr}
(A C_{xz}B C_{zx}).
\]

Applying

\[
A=B=\frac{P}{n-1}
\]

gives

\[
\boxed{
\operatorname{Cov}(q_i,q_j)
=
\frac{2}{(n-1)^2}
\operatorname{tr}(P C_{ij}P C_{ji}).
}
\]

For \(i=j\), this reduces to the merged single-scale map-variance theorem.

### Quadratic-storage identity

The implementation avoids explicitly multiplying dense centering matrices. With \(C_{ji}=C_{ij}^T\) and \(M=C_{ij}\),

\[
\operatorname{tr}(PMPM^T)
=
\|M\|_F^2
-
\frac{\|M\mathbf1\|^2+\|M^T\mathbf1\|^2}{n}
+
\frac{(\mathbf1^TM\mathbf1)^2}{n^2}.
\]

This reduces the moment calculation from cubic to quadratic cost after the covariance block is constructed.

## 4. Bias-corrected covariance

Define the deterministically corrected estimators

\[
\tilde q_i=\frac{q_i}{b_i}.
\]

Then

\[
\mathbb E[\tilde q_i]=V_i
\]

and

\[
\boxed{
\operatorname{Cov}(\tilde q_i,\tilde q_j)
=
\frac{\operatorname{Cov}(q_i,q_j)}{b_i b_j}.
}
\]

The correction removes the declared mean bias but preserves the finite-map information limit.

## 5. First-order logarithmic covariance

For positive \(\mu_i\), first-order propagation gives

\[
\log q_i
\simeq
\log\mu_i+rac{q_i-\mu_i}{\mu_i}.
\]

Therefore

\[
\boxed{
[C_{\log q}]_{ij}
\simeq
\frac{\operatorname{Cov}(q_i,q_j)}{\mu_i\mu_j}.
}
\]

The same covariance follows for \(\log\tilde q_i\), because deterministic multiplicative correction contributes only an additive constant:

\[
\log\tilde q_i=\log q_i-\log b_i.
\]

This is a delta-method covariance. The correlated quadratic forms are not asserted to be exactly jointly log-normal.

The single-scale moment-matched degree of freedom is

\[
\nu_{i,\mathrm{eff}}
=
\frac{2}{[C_{\log q}]_{ii}}.
\]

It is a moment summary, not an exact chi-square distributional claim.

## 6. Fisher information with same-raster covariance

For the isotropic two-dimensional Gaussian benchmark,

\[
V(s)=\frac{A}{1+2s^2/\xi^2}.
\]

Use logarithmic parameters

\[
\theta=(\log A,\log\xi).
\]

The log-variance Jacobian row at scale \(s_i\) is

\[
J_i
=
\left(
1,
\frac{4(s_i/\xi)^2}{1+2(s_i/\xi)^2}
\right).
\]

With the full same-raster log covariance,

\[
\boxed{
F_{\rm map}
=
J^T C_{\log q}^{+}J.
}
\]

The commonly used but incorrect independent-pixel approximation assigns

\[
C_{\rm nominal}
=
\frac{2}{n-1}I_m,
\]

so

\[
F_{\rm nominal}=J^T C_{\rm nominal}^{-1}J.
\]

This approximation makes two errors:

1. it replaces each scale's small effective degree of freedom with \(n-1\);
2. it deletes the cross-scale covariance generated by observing the same field.

## 7. Controlled three-scale raster

Declare

\[
A=(0.005)^2,
\qquad
\xi=2\ \mu\mathrm m,
\]

with

\[
s=[0.23094,1.15470,4.61880]\ \mu\mathrm m.
\]

Use one \(10\times10\) raster with \(0.5\ \mu\mathrm m\) spacing at all three scales.

The expected naive map variances relative to the marginal filtered variances are

\[
\boxed{
b=[0.5281,0.4019,0.0830].
}
\]

The first-order log-estimator standard deviations are

\[
\boxed{
\sigma_{\log q}
=[0.6949,0.7747,0.9553].
}
\]

For comparison, treating 100 pixels as independent gives

\[
\sqrt{\frac{2}{99}}=0.1421
\]

at every scale.

The same-raster cross-scale correlations are

\[
\boxed{
\rho=
\begin{pmatrix}
1 & 0.9460 & 0.1800\\
0.9460 & 1 & 0.2884\\
0.1800 & 0.2884 & 1
\end{pmatrix}.
}
\]

The two smallest probes are especially correlated because they view nearly the same spatial modes.

Using the full log covariance rather than nominal independent-pixel counting gives parameter-standard-deviation inflation

\[
\boxed{
\frac{\sigma_{\log A,\rm map}}
{\sigma_{\log A,\rm nominal}}
=4.44,
}
\]

and

\[
\boxed{
\frac{\sigma_{\log\xi,\rm map}}
{\sigma_{\log\xi,\rm nominal}}
=2.87.
}
\]

The determinant of the two-parameter covariance increases by

\[
\boxed{404.7}.
\]

Thus nominal pixel counting understates the joint uncertainty volume by more than two orders of magnitude in this controlled dense-map design.

## 8. Fixed-field-of-view oversampling

Compare approximately the same field of view:

```text
10 by 10 pixels at 0.5 um spacing
20 by 20 pixels at 0.25 um spacing
```

The nominal count increases from 100 to 400. The full-covariance parameter standard deviations change only from approximately

```text
SD(log A):   0.5766 -> 0.5745
SD(log xi):  0.3231 -> 0.3218
```

which is less than a 0.4% improvement.

The independent-pixel model instead predicts nearly the exact factor-of-two improvement associated with quadrupling independent sample count.

The false joint-covariance-volume optimism therefore becomes more severe as the same finite field of view is sampled more densely.

## 9. Measurement-design consequence

A multiresolution raster should be analyzed by one of the following routes:

1. use independent specimen regions or independent maps at each allocated repeat;
2. retain the full block covariance among scales and spatial positions;
3. propagate the exact quadratic-form covariance of map summary statistics;
4. report effective degrees of freedom and bias factors as diagnostics, not as substitutes for the full likelihood when precision matters.

Nominal pixel count is not a valid repeat count for the allocation theorem. Multiple probe scales on the same field are not independent observations.

## 10. Claim boundary

Gaussian quadratic-form cross moments, delta-method propagation, and Fisher information are established mathematics.

The R04 contribution is the explicit finite-resolution, multiscale HgCdTe-map consequence and its integration with the measurement-kernel, covariance-family, calibration, and allocation hierarchy.

The result does not identify a specimen covariance, make the log-quadratic distribution Gaussian, prescribe a universal map pitch, or authorize a manuscript.
