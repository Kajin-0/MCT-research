\section{Limitations}
\label{sec:limitations}

\begin{enumerate}
  \item The experimental application contains two 300~K spectra from one source study.
  \item The data were recovered from calibrated primary-figure bitmaps rather than native instrument exports.
  \item Specimen-level composition uncertainty is unreported.
  \item Carrier type and density are unreported.
  \item Several fitted candidates are boundary-limited and are not interpreted as identified edges.
  \item The highest fixed threshold fails the coordinate-sensitivity gate for one specimen.
  \item The observation decomposition is local and diagnostic; it is not a universal physical correction model.
  \item The acquisition-design minimum is established only within the declared two-level candidate family and audit gates.
  \item Independent low-temperature and cross-laboratory validation remain desirable strengthening evidence but are not used as fit authority here.
\end{enumerate}
