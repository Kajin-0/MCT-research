# Decision record: static CdTe Kane extraction audit

**Date:** 2026-07-18  
**Status:** controlling audit record; implementation corrections pending  
**Scope:** static CdTe finite-k Kane extraction, prior-art boundary, model-reduction claims, and computation authorization  
**Evidence source:** archived `cdte-irrep-symmetry-probe-evidence` artifact from the merged static-method smoke chain

## Decision summary

The static sprint produced a robust linear validation result, but its quadratic interpretation used the wrong effective-Hamiltonian construction.

The repository's all-state overlap formula approaches the bare fixed-basis projection

```text
P_Gamma H(k) P_Gamma
```

as the retained eigenstate set becomes complete. It does not approach an isospectral eight-band Löwdin/downfolded Hamiltonian. The resulting error begins at quadratic order, so the dominant linear couplings remain sound while the reported quadratic coordinates and model-selection frontier are materially distorted.

The scientific default must be changed to a selected-eight-band polar/parallel-transport Hamiltonian that preserves the selected DFT eigenvalues exactly.

No additional static electronic-structure calculation is authorized until this post-processing correction and its gauge audit are implemented.

## 1. Prior-art correction

The nearest prior method is DFT2kp:

- Cassiano et al., *DFT2kp: effective k.p models from ab-initio data*, SciPost Physics Codebases 25 (2024), arXiv:2306.08554.
- It extracts Kane and Luttinger parameters from Quantum ESPRESSO wavefunctions, performs symmetry adaptation, and constructs zincblende models for GaAs, CdTe, and HgTe.
- Its published CdTe model contains four independent linear and ten independent quadratic coefficients.

Related prior methods include:

- Jocić and Vukmirović (2020): ab-initio symmetry-adapted k.p Hamiltonians and unitary transformation into a canonical basis;
- VASP2KP (2023): automatic symmetry, momentum, time-reversal, k.p, and g-factor extraction;
- general symmetry-adapted model construction and fitting from complete multipole/invariant bases.

Consequences:

1. first-principles symmetry-adapted k.p extraction is established;
2. complete four-linear/ten-quadratic zincblende Hamiltonians are established;
3. published CdTe coefficients already exist;
4. the repository cannot claim a new complete static CdTe Hamiltonian;
5. useful contribution must be narrower: cross-method reproduction, reduced-model diagnostics, observable consequences, or a genuinely new finite-temperature full-matrix result.

## 2. Complete linear invariant space

The repository's exact Novik `Gamma6 + Gamma8 + Gamma7` representation gives

```text
dim H_linear(Td + time reversal) = 4.
```

The four one-dimensional sectors are:

```text
Gamma6-Gamma8
Gamma6-Gamma7
Gamma8-Gamma8
Gamma8-Gamma7
```

There are no allowed linear terms in `Gamma6-Gamma6` or `Gamma7-Gamma7`.

The existing two-`P` model includes only the first two sectors. Fitting the archived finite-k matrices gives:

```text
reduced two-P training residual     = 8.2018e-3
complete 4D training residual       = 1.65e-7
complete 4D [110] holdout residual  = 1.72e-7
```

The dominant couplings are approximately:

```text
P8 = 6.77222 eV A
P7 = 6.20183 eV A
```

They agree with the DFT2kp CdTe values after convention conversion at roughly the one-percent level.

### Controlling interpretation

The old phrase "two-P closure" is inaccurate.

Use:

> The inversion-symmetric two-P subspace captures approximately 99.18% of the linear matrix norm. The complete four-invariant zincblende space, including the two bulk-inversion-asymmetry sectors, closes the matrices at the numerical differentiation floor.

The four-dimensional structure is prior art. The repository result is an independent finite-k reproduction and a validation of its canonical-basis machinery.

## 4. Root cause in the finite-k reconstruction

The current production formula is

```text
M_N = S_N S_N^dagger
U_N = M_N^(-1/2) S_N
H_N = U_N E_N U_N^dagger
```

where `S_N` contains overlaps between the fixed Gamma reference basis and `N` finite-k eigenstates.

In the complete-state limit,

```text
S_infinity S_infinity^dagger = I
S_infinity E_infinity S_infinity^dagger = P_Gamma H(k) P_Gamma
```

and therefore

```text
H_infinity = P_Gamma H(k) P_Gamma.
```

This is the bare Hamiltonian block in the fixed Gamma basis. It is not generally isospectral to the selected low-energy bands and does not contain the complete second-order remote-band renormalization required of an effective Hamiltonian.

### Two-level counterexample

For

```text
H(k) = [[0, v k], [v k, Delta]],
```

the exact low-energy band has

```text
E_-(k) = -(v^2/Delta) k^2 + O(k^4),
```

while the complete bare projection onto the first basis state is zero. Thus a complete-state spectral projection can erase the very second-order shift the effective Hamiltonian is intended to reproduce.

## 5. Direct eigenvalue falsification

Using the archived CdTe evidence, the all-40-state reconstruction misses the selected bands by:

```text
[001], |k|=0.01 A^-1 : 1.030 meV
[111], |k|=0.01 A^-1 : 0.795 meV
[110], |k|=0.01 A^-1 : 0.552 meV
```

At half radius, the errors are approximately one quarter as large. The coarse/fine ratios are approximately four, proving that the reconstruction error is quadratic in `k`.

The error is therefore of exactly the same order as the extracted quadratic Hamiltonian.

The existing code computes a `projected_vs_selected_eigenvalue_residual_ev` diagnostic but does not gate on it. A scientific effective-Hamiltonian reconstruction must fail closed when it is not isospectral at the declared precision.

## 6. Correct selected-band polar construction

Let `B(k)` be the square 8x8 overlap matrix between the fixed Gamma reference basis and the selected finite-k bands 31-38. Define

```text
W(k) = [B(k) B(k)^dagger]^(-1/2) B(k)
H_eff(k) = W(k) E_selected(k) W(k)^dagger.
```

`W` is unitary, so

```text
eigenvalues[H_eff(k)] = E_selected(k)
```

exactly.

Remote-band physics is already encoded in the selected eigenvalues and selected eigenvectors. Explicitly inserting all remote eigenstates into the spectral reconstruction is not required and, in the complete limit, changes the target object into the bare projection.

This selected-band polar construction is the scientific default authorized by this decision record.

## 9. Gauge-dependence boundary

The selected-band polar Hamiltonian is isospectral, but an effective Hamiltonian is not unique under a smooth k-dependent unitary transformation inside the selected manifold:

```text
H_eff(k) -> V(k)^dagger H_eff(k) V(k).
```

Band energies are invariant; individual matrix coefficients need not be.

The present polar construction defines a specific parallel-transport-like gauge anchored to the canonical Gamma basis. Before treating individual quadratic coordinates as unique material parameters, the repository must:

1. state the gauge definition explicitly;
2. verify symmetry and time-reversal covariance of that gauge;
3. quantify coefficient variation under allowed smooth gauge transformations;
4. distinguish gauge-invariant observables and subspace distances from gauge-dependent coordinates;
5. compare with DFT2kp only after an explicit convention/gauge mapping.

Until then, the robust conclusion is reduced-model failure in the declared gauge and improved reproduction of selected observables—not universal uniqueness of every coefficient.

## 11. Separate issues discovered during the audit

### Matrix covariance representation

An 8x8 Hermitian matrix has 64 independent real degrees of freedom. The current 128-real-component representation duplicates Hermitian-conjugate information and leads to a singular exact covariance. Deterministic Frobenius fits remain valid, but statistical degrees of freedom, reduced chi-square, standard errors, and likelihood claims require a 64-dimensional independent-Hermitian schema.

### Seiler held-out wording

The Seiler analysis profiles one additive offset for each held-out specimen. It tests conditional thermal-shape transfer, not absolute prediction of the specimen's gap. Results should be labeled accordingly.

### A0 first-point response

PR #98 remains an execution/provenance success but a physical-response sanity failure. Its raw negative acoustic triplet, Born-charge sum violation, large ASR optical shift, and fixed-volume pressure must be recorded before any second response point. This workstream is independent of the static Kane correction.
